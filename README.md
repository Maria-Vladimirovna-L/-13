# Currency Converter

**Автор:** Каверзин Глеб Константинович
**Описание:** Простое GUI-приложение для конвертации валют с использованием внешнего API и сохранением истории.

## Требования

- Python 3.8+
- Библиотеки: `tkinter`, `requests`
  Установить: `pip install requests`

## Как получить API-ключ

1. Зарегистрируйтесь на сайте [exchangerate-api.com](https://www.exchangerate-api.com/).
2. Получите бесплатный или платный API-ключ.
3. Вставьте ключ в строку `API_KEY` в файле `main.py`.
   ```python
   API_KEY = 'YOUR_API_KEY'
   ```

## Запуск приложения

1. Убедитесь, что Python установлен.
2. Скачайте файлы проекта.
3. Запустите:
   ```
   python main.py
   ```
4. В открывшемся окне выберите валюты, введите сумму и нажмите «Конвертировать».
5. История операций сохраняется в файл `history.json`.

## Примеры использования

- Конвертировать 100 USD в EUR.
- Проверить курс GBP к RUB.
- Посмотреть историю последних 20 операций.
