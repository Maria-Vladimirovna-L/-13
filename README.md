# 💱 Currency Converter

**Автор:** Каверзин Глеб Константинович

Готовый проект GUI-приложения для конвертации валют с использованием внешнего API, сохранением истории операций в файл и интеграцией с системой контроля версий Git.

## 🛠️ Требования

*   **Язык:** Python 3.8 или новее.
*   **Библиотеки:** `requests` (установить: `pip install -r requirements.txt`).
*   **API:** Аккаунт на [exchangerate-api.com](https://www.exchangerate-api.com/) для получения бесплатного ключа.

## 🚀 Как запустить

1.  **Получите API-ключ:**
    Зарегистрируйтесь на сайте [exchangerate-api.com](https://www.exchangerate-api.com/) и получите бесплатный ключ.
2.  **Настройте ключ в коде:**
    Откройте файл `main.py` и замените строку `API_KEY = "ВАШ_API_КЛЮЧ"` на ваш реальный ключ.
3.  **Установите зависимости:**
    В терминале выполните команду:
    ```bash
    pip install -r requirements.txt
    ```
4.  **Запустите приложение:**
    Выполните команду в папке с проектом:
    ```bash
    python main.py
    ```

## ✨ Возможности приложения

*   **Выбор валют:** Конвертация между популярными валютами (USD, EUR, GBP, JPY, CAD, AUD, CHF, CNY, RUB).
*   **Проверка ввода:** Сумма должна быть положительным числом.
*   **Актуальные курсы:** Данные берутся с внешнего API в реальном времени.
*   **История операций:** Все расчеты автоматически сохраняются в файл `history.json`.
*   **Просмотр истории:** Таблица с датой, операцией и результатом отображается в окне программы.
*   **Очистка истории:** Кнопка для удаления всех записей из истории.

## ⚠️ Примечания

*   Бесплатный тариф API позволяет до 1500 запросов в месяц.
*   История сохраняется локально в файле `history.json`.
*   Для расширения функционала можно добавить экспорт истории в CSV или поддержку большего числа валют.
